# Widgets
**Deine Legosteine**

Hier erstellst du deine eigenen, kleinen Bausteine. Anstatt denselben Code (z. B. für einen formatierten Button) überall zu kopieren, schreibst du ihn hier einmal auf.

- **Was kommt hier rein?** Wiederverwendbare UI-Elemente.
- **Beispiel:** `benutzer_karte.dart`, `mein_standard_button.dart`, `custom_text_feld.dart`.
- **Goldene Regel:** Ein Widget sollte idealerweise nur **eine** Aufgabe haben und so gebaut sein, dass man es an vielen Stellen einsetzen kann.

*Tipp: Je öfter du ein Design-Element in deiner App benutzt, desto eher sollte es ein eigenes Widget in diesem Ordner sein.*
