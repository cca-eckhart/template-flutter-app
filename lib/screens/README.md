# Screens
**Deine App-Seiten**

Hier liegen die großen Bausteine deiner App. Jede Datei in diesem Ordner stellt in der Regel einen **ganzen Bildschirm** dar, den ein Benutzer sehen kann.

- **Was kommt hier rein?** Widgets, die ein `Scaffold` beinhalten (mit AppBar, Body, etc.).
- **Beispiel:** `login_screen.dart`, `home_screen.dart`, `profil_seite.dart`.
- **Goldene Regel:** Nutze Screens, um die verschiedenen Bereiche deiner App voneinander zu trennen.

*Tipp: Wenn ein Screen zu lang wird (über 200 Zeilen), lagere Teile davon in den `widgets/` Ordner aus!*
