# Models
**Der Bauplan deiner Daten**

In diesen Ordner kommen alle Dateien, die beschreiben, wie deine **Objekte** aussehen. Hier gibt es kein Design (keine Farben, kein Padding), sondern nur reine Informationen.

- **Was kommt hier rein?** Einfache Dart-Klassen.
- **Beispiel:** Eine Klasse `Student` mit `name`, `alter` und `klasse`.
- **Goldene Regel:** Wenn du Daten speichern oder von einer API laden willst, erstellst du hier zuerst das passende Model.

*Tipp: Modelseiten fangen meistens nicht mit "Widget" an, sondern sind reine Datenstrukturen.*
